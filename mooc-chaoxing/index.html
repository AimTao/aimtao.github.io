刷学习通等各类学习软件视频

本文最后更新于：1 年前

时近年底，各种学习软件的视频任务都将接近截止日期，天天拿着手机刷视频，难道是想让我多买几部手机？

视频多，不能快进，没有倍速，费时费力，但无奈于要算入考核成绩中。以上为背景。

有什么解决方案？

方案一：增加快进和进度条拖拽功能。

考虑到有些视频中间还有题目，不便操作；而且长时间拖拽进度条会导致操作频繁，锁定账号。此方案pass。

方案二：增加倍速播放功能。

由于目前大多数 学习 视频网站都是 HTML5 播放器，（毕竟 Chrome 2020年就不支持 Flash 了）而 HTML5 中可以通过 js 来控制 html 的 <video> 元素 的播放速度。

按 F12 ，打开console（控制台） –> 复制下面 js 代码 –> 回车。

原理如下：（无需求可跳过）

1. document

在 HTML DOM (Document Object Model) 中 ，一个 HTML文档 是一个 文档节点，当文档节点被浏览器载入就会成为 Document对象。通过 Document对象 可以从脚本中对 HTML 页面中的所有元素进行访问。

2. querySelector(‘video’)

querySelector() 方法返回 Document对象 中匹配指定 CSS 选择器的一个元素。故 document.querySelector('video') 就是返回 <video> 元素，

3.defaultPlaybackRate

Video defaultPlaybackRate属性，设置播放速度，想代码体验一下点这里。

可不可以优雅些呢？

  • 学习通的视频分的细碎细碎的，一直打开 console 复制粘贴代码，着实不太优雅。
  • 鼠标的光标不能离开播放窗口，否则会暂停，也就是说不能用电脑干别的了。

改进一：Chorme 插件

Github上有个项目，一款专为Html5播放器加速的插件，videospeed。

在 Chrome商店 安装，最高16倍速播放，11分钟的视频40秒搞定！

PS. 插件网站推荐

有同学反应，不会fq，打不开 Chrome商店 ，作为根正苗红的好青年，该如何获取 chorme 插件呢？

访问 这个网站 crxdl.com 即可。

下载后，打开 chrome -> 更多工具 -> 扩展程序，右上角打开开发者模式，将下载的压缩包中的 crx 文件拖入浏览器就好了。

改进二：虚拟机解放鼠标

自从 廖老板用python代码开始刷阳光体育跑步 开始，我就觉得，没有什么偷懒的事是不可能的，除非你还不够懒。

如何才能解放鼠标呢？我想到了最近经常用的虚拟机，虚拟机的鼠标是虚拟机的，外部的鼠标是外部的，互不干扰，也就是说让视频自己在虚拟机中全屏播放，最小化虚拟机窗口就可以了。

  • Mac推荐：VMware Fusion
    • pro 版激活码：7HYY8-Z8WWY-F1MAN-ECKNY-LUXYX
  • windows推荐：VMware workstation
  • 镜像推荐：itellyou
